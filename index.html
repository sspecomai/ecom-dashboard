🔗 Connect Google Sheets

You need 3 sheets, each published separately as CSV.
Google Sheets → File → Share → Publish to web → Select sheet tab → CSV → Publish → Copy URL.

Sheet 1 — RawData (daily transactions, required)
Columns: date, channel, type, brand, category, orders, revenue, budget_revenue, ly_revenue, lm_revenue, sessions, budget_sessions, cogs, discount_amount, fulfillment_cost, marketing_spend, returns, cancellations, new_customers, returning_customers

Sheet 2 — TeamKPIs (updated monthly, optional)
Columns: period, team, metric, value, target, lm_value, ly_value, unit, status

Sheet 3 — CampaignData (marketing campaigns, optional)
Columns: date, campaign, channel, spend, impressions, clicks, orders, revenue, new_customers

Sheet 4 — InventorySTR (sell-through & stock, optional)
Columns: sku, product_name, brand, category, channel, opening_stock, units_sold, closing_stock, target_str, lm_str, ly_str, cost_price, selling_price, stock_value, oos_flag, slow_mover_flag
Columns: sku, product_name, brand, category, channel, opening_stock, units_sold, closing_stock, target_str, lm_str, ly_str, cost_price, selling_price, stock_value, oos_flag, slow_mover_flag
